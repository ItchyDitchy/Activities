# Special Task 1

## Instructions

Write a program that will perform the four (4) types of built-in functions:
- SRAND
- ROUND OFF
- POWER
- SQUARE ROOT

## Objectives

A. Use function overloading approach.  
B. WIth a main menu appeared on screen in first run.

### Example:

```txt
SELECT BUILT IN FUNCTIONS TO PERFORM

1. SRAND
2. ROUND OFF
3. POWER
4. SQUARE ROOT
5. EXIT
```
